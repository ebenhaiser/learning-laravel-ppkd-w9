MVC:
- model      : database
- view       : html, css, or javascript
- controller : menghubungkan antara model dan view

artisan      :
composer     :
method       : artisan kecakapan/kebisaan

    - put    : mengirim data dari form (update)
    - delete : mengirim data dari form (delete)
visibililasi : private, public, static
